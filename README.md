# HR Data Analysis Dashboard

## Project Overview
An end-to-end HR data analysis project built in Microsoft Excel 
using Power Query for data cleaning and Pivot Tables for analysis.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables and Pivot Charts
- Power Pivot Data Model

## Key Insights
- Total workforce: 200 employees
- Active employees: 119 (59.50%)
- Attrition rate: 40.50% — critically high
- Average salary: ₦720,000
- Average performance score: 2.9 out of 5
- Gender split: 51.50% Male, 48.50% Female
- Hiring peaked in 2020 and declined sharply through 2023

## Dataset
- 203 rows | 14 columns
- Cleaned using Power Query
- Relationships built using Power Pivot Data Model
